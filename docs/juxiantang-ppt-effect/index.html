NARRATIVE LOGIC
汇报逻辑总览

先证明为什么需要它，
再证明为什么只能是聚贤堂。

01 市场有空位 高端太远，日常太薄，潮流太快。北京缺一个有底子、又能日常进入的京鲁菜品牌。
02 生意要重定 如果只卖菜，就会同质化；聚贤堂要做的是承接人生场合的一处局。
03 品牌有证据 八大堂、老舍婚宴、炸响铃、堂会空间，不是包装概念，而是可转译的资产。
04 体验能落地 馆、味、堂分别对应店型、产品和空间礼数，形成可复制的顾客体验。
05 模型可生长 先用菜馆跑通基本盘，再以饭庄扩面，最后用堂会建立品牌上限。

AN OBSERVATION
一个判断

北京餐桌上，
中间是空的。

聚贤堂要占住的不是高端会所，也不是普通京菜，而是一个有老字号底子、有文化厚度、又让普通人敢进的位置。
高端会所 庄重、正式、客单高。适合宴请，但离日常太远。
日常京菜 亲切、大众、频次高。好进入，但品牌记忆不深。
潮流主题餐饮 话题强、传播快。容易热闹，也容易过时。

POSITIONING SYSTEM
定位系统

小馆有堂气，
不是一句口号，是三件事同时成立。

馆

先让人敢进

200-300 平方米，商场/社区可复制，人均约 100。小馆决定进入门槛和经营模型。

味

再让人记住

炸响铃双汁、文武烤鸭、京鲁闾巷菜。味道不是堆菜单，而是建立可传播的招牌锚点。

堂

最后让人愿意带客

牌匾、入席、上菜报名、包间题名。堂气负责体面、谈资和场合感。

BRAND CREDO
品牌信条

宴饮有道，
贤者自来。

01宴饮有道不是堆砌，是规矩。从入席、奉茶、上菜、敬酒到散席，每一步都有自己的腔调。

EIGHT HALLS
八大堂资产

堂号不是装饰，
是聚贤堂的身份系统。

清末民初，北京饭庄的最高规格是庭院、戏台、宴席和雅集。聚贤堂要从这套“堂”的秩序里，转译出今天用户能进入的餐饮体验。

PLACE
报子街记忆以地名和旧城记忆作为文化坐标，而不是虚构背景。
HALL
三面有楼，内设戏台为空间叙事提供“入堂、见戏、入席”的结构原型。
CUISINE
经营鲁菜让京鲁菜馆定位具备历史来处，不只是当代品类选择。
SIGNATURE
炸响铃双汁让文化资产能被用户点到、吃到、拍到、讲出来。

BRAND MOTIF
品牌母题

老舍婚宴，

EVENT

SCENE
婚礼与宴席把品牌从“吃饭”拉到“人生场合”的层面。
MEMORY
城市共同记忆外地客听得懂，本地客也觉得有来处。
TRANSFER
今天的局生日、周年、庆功、小宴，都可以承接这条母题。

ENTERING SEQUENCE
入堂六步
