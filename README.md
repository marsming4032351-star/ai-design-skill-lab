# Design Data Factory v6

## 🚀 What is this?

Design Data Factory is an AI-powered system that transforms design workflows into reusable data assets.

Instead of treating design as static output, this system builds a closed loop:

ingest → run → critic → archive → pattern reuse

---

## 🧠 Core Idea

Design is no longer a file.
Design is a data flow.

---

## 🔁 Core Loop

- **Ingest**: structure raw assets into data
- **Run**: generate design concepts
- **Critic**: evaluate quality using rules
- **Archive**: extract reusable patterns
- **Reuse**: apply patterns automatically in future runs

---

## 🔥 Key Features

- Pattern auto-generation & recommendation
- Reusable design intelligence
- Structured design pipeline
- AI-operable system via [Codex 操作指南](docs/CODEX_GUIDE.md)
