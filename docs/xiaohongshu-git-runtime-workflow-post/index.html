02 / 05

commit 是本地历史，push 是外部同步

git commit

把这次改动固定成本地快照。它是 Runtime milestone，不等于公开。

git push

把本地历史同步到 GitHub。它会进入团队、CI、开源和长期记录。

AI 可以 commit，但不应该默认自动 push。
本地记忆 != 云端发布review before push
