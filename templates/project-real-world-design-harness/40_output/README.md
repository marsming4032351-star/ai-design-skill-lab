# 40_output：视觉与交付输出

这里存放通过 framework gate 之后生成的视觉提示词、图像、HTML、PPT、图文包和交付材料。

## 应该放什么

- 基于设计框架生成的 visual prompts。
- 效果图、空间图、品牌图、门头图、展项图。
- HTML deck、小红书图文包、提案稿、发布文案。
- 每个输出对应的框架依据说明。

## 不能做什么

- 不要脱离框架追求表面美感。
- 不要把未经评审的图当最终方案。
- 不要只保存图片，不保存提示词和来源。

## 阶段产出

- `visual-prompts.md`
- `output-index.md`
- `presentation/`
- `images/`
- `social-post/`
