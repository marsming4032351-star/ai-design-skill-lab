# 10_research：研究阶段

这里存放 Research Agent 和人工研究整理出的事实、背景和资料缺口。

## 应该放什么

- 场地事实表。
- 历史、街区、行业、品牌和竞品研究。
- 人群和使用场景摘要。
- 资料来源、引用链接、访谈摘要。
- 需要补充验证的问题。

## 不能做什么

- Research Agent 不能直接给设计方案。
- 不要把推断写成事实。
- 不要用风格词代替研究结论。

## 阶段产出

- `site-research.md`
- `history-context.md`
- `audience-research.md`
- `competitor-research.md`
- `research-gaps.md`
