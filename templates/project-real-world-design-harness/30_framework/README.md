# 30_framework：设计框架

这里存放进入视觉生成之前必须完成的设计框架。

## 应该放什么

- 设计定位。
- 核心命题。
- 关键词。
- 空间策略。
- 视觉策略。
- 运营约束。
- 不做什么。
- 进入 Visual Agent 的准入说明。

## 不能做什么

- 不要只写抽象口号。
- 不要用“高级、年轻、自然、科技感”等空泛词替代策略。
- 不要在框架通过前放效果图提示词。
