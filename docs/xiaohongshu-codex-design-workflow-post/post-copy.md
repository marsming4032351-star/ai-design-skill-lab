# 小红书发布文案：用 Codex 做设计策划工作流

## 标题

我用 Codex + GPT Image 跑通了一套设计策划工作流

## 正文

这次不是单纯让 AI “写一篇策划案”，而是把 Codex 当成一个可以调用工具的设计策划工作台。

我的完整流程是：

1. 用 `markitdown` 读取 PPT / PDF / Word 资料
2. 用 `brainstorming` 先做策略头脑风暴
3. 让 Codex 拆出品牌定位、产品结构、视觉资产和 PPT 逻辑
4. 用 GPT Image / Codex Image 生成主视觉和图文氛围
5. 用 HTML/CSS 叠加中文排版，避免图片中文字乱码
6. 用 `playwright` 打开页面截图，检查能不能发布
7. 最后输出 PNG 图文卡片和发布文案

我现在更认可一种 AI 工作方式：

不是一个 prompt 直接出结果，而是让 Codex 调用不同 skills，把资料、策略、视觉和交付串成一条工作流。

尤其适合：

- 品牌策划
- PPT 提案重构
- 空间/餐饮品牌定位
- 小红书图文教程
- 设计方法论沉淀

这套图里的主视觉就是用 GPT Image 在 Codex 里生成的，再用代码做中文排版，所以既有图像质感，也能保证文字准确。
