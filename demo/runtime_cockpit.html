The value is the lifecycle around generation: planning, critique, retry, review, archive, and memory.

Pause is a feature

当前能力

M1

Skeleton

TaskSpec, RunContext, StepResult, SkillRegistry, ReviewDecision, and the runtime shell.

M2

Runtime Execution

Deterministic planner, generator, critic, review, and archive lifecycle.

M3

Review / Retry Loop

Critic feedback returns to the generator until pass or max retry exhaustion.

Boundary

Blocked State

WAITING_AUTH, WAITING_HUMAN, BLOCKED_EXTERNAL, PAUSED, and RESUMABLE as design states.

Workspace

Feishu Workspace

Guides and reports are published to Feishu for mobile reading and stakeholder review.

Demo 运行命令

python3 scripts/run_harness_demo.py examples/harness_goal.yaml

Run from the repository root. The demo uses deterministic mock stages and does not call a live LLM.

输出解释

run_id本次运行的唯一编号。
final_stateARCHIVED, FAILED, or REVIEW_REQUIRED.
step_history每个阶段返回的 StepResult。
event_logRuntime 决策与事件轨迹。
review_required是否声明需要 review gate。
retry_count已启动的重试次数。
max_retries允许的最大重试次数。
