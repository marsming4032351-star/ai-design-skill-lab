NARRATIVE LOGIC
汇报逻辑总览

先证明为什么需要它，
再证明为什么只能是聚贤堂。

01 市场有空位 高端太远，日常太薄，潮流太快。北京缺一个有底子、又能日常进入的京鲁菜品牌。
02 生意要重定 如果只卖菜，就会同质化；聚贤堂要做的是承接人生场合的一处局。
03 品牌有证据 八大堂、老舍婚宴、炸响铃、堂会空间，不是包装概念，而是可转译的资产。
04 体验能落地 馆、味、堂分别对应店型、产品和空间礼数，形成可复制的顾客体验。
05 模型可生长 先用菜馆跑通基本盘，再以饭庄扩面，最后用堂会建立品牌上限。

AN OBSERVATION
一个判断

北京餐桌上，
中间是空的。

聚贤堂要占住的不是高端会所，也不是普通京菜，而是一个有老字号底子、有文化厚度、又让普通人敢进的位置。
高端会所 庄重、正式、客单高。适合宴请，但离日常太远。
日常京菜 亲切、大众、频次高。好进入，但品牌记忆不深。
潮流主题餐饮 话题强、传播快。容易热闹，也容易过时。

POSITIONING SYSTEM
定位系统

小馆有堂气，
不是一句口号，是三件事同时成立。

馆

先让人敢进

200-300 平方米，商场/社区可复制，人均约 100。小馆决定进入门槛和经营模型。

味

再让人记住

炸响铃双汁、文武烤鸭、京鲁闾巷菜。味道不是堆菜单，而是建立可传播的招牌锚点。

堂

最后让人愿意带客

牌匾、入席、上菜报名、包间题名。堂气负责体面、谈资和场合感。

05 / 11
CULTURAL PROOF
文化证据

文化不是装饰，
是聚贤堂进入市场的理由。

如果没有真实资产，“小馆有堂气”会变成空泛的新中式表达。聚贤堂的优势在于，每个体验动作都能找到来历。

ASSET 01
八大堂 提供堂号身份、饭庄规格和老北京宴饮体系的历史底座。
ASSET 02
老舍婚宴 证明聚贤堂从一开始就不是普通吃饭之地，而是承接人生场合的地方。
ASSET 03
炸响铃双汁 把历史来历转成可点、可拍、可讲述的招牌产品。
ASSET 04
堂会礼数 把空间、服务和话术变成体验系统，让一百块的店也有入席感。
