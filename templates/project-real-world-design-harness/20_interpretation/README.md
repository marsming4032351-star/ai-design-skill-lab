# 20_interpretation：解释与机会

这里存放从研究资料中提炼出的矛盾、情绪、甲方动机和设计机会。

## 应该放什么

- 关键矛盾列表。
- 情绪判断和现场体验解释。
- 甲方真实动机分析。
- 用户机会、空间机会、品牌机会、运营机会。
- 机会优先级和取舍说明。

## 不能做什么

- 不要只复述研究资料。
- 不要直接跳到视觉风格。
- 不要把所有想法都保留，必须做取舍。
