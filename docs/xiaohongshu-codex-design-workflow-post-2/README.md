# 小红书续篇发布包：Codex 入口和小白上手路径

这是 `docs/xiaohongshu-codex-design-workflow-post/` 的续篇图文包。

上一篇展示 Codex + GPT Image 设计策划工作流，这一篇面向技术基础较弱但需求明确的终端用户，重点解释：

- Codex 有终端版和桌面端 / ChatGPT 入口；
- 小白应该如何选择入口；
- 不懂代码时如何先用话术把需求交给 Codex；
- 为什么关注、点赞、收藏会帮助后续获得模板、案例和实操拆解。
