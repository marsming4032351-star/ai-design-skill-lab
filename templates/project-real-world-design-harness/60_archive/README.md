# 60_archive：归档复用

这里存放通过评审后值得长期保留和复用的资产。

## 应该放什么

- 通过验证的设计框架。
- 可复用 Pattern。
- Prompt 变体。
- 评审标准和 checklist。
- 项目复盘。

## 不能做什么

- 不要归档未经评审的效果图。
- 不要归档没有来源、没有判断、没有适用边界的素材。
- 不要把一次性草稿当成长期资产。

## 阶段产出

- `project-retrospective.md`
- `reusable-pattern.md`
- `prompt-variants.md`
- `rule-update-suggestions.md`
