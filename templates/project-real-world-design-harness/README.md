# Real-World Driven Design Harness 项目模板

这个模板用于真实世界驱动的设计项目。它要求先完成现场输入、研究解释和设计框架，再进入视觉生成，避免项目过早变成“AI 效果图生产”。

## 目录结构

```text
00_input/
10_research/
20_interpretation/
30_framework/
40_output/
50_review/
60_archive/
```

## 使用原则

- 先放现实输入，再做研究。
- 先形成设计机会，再写框架。
- 先通过 framework gate，再进入视觉提示词和效果图。
- 所有输出都要能追溯到场地、历史、人群、甲方目标或运营约束。
