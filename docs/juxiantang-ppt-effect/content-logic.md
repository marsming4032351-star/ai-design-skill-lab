# 聚贤堂 PPT 完整内容逻辑

## 核心叙事

这套 PPT 的任务不是单纯展示“视觉高级”，而是说服甲方：聚贤堂有明确市场位置、有真实文化资产、有可落地的产品和空间系统，也有可分阶段推进的经营路径。

主线：

> 先证明为什么需要它，再证明为什么只能是聚贤堂。

最终记忆点：

> 聚贤堂：小馆有堂气。一座楼下能进的八大堂。

## 必须保留的因果链

1. **市场缺位**：高端会所太远，日常京菜太薄，潮流主题餐饮太快。
2. **品牌位置**：聚贤堂进入的是“有底子但敢进”的中间层。
3. **战略定义**：如果只是卖京鲁菜，会陷入价格、出菜、翻台的同质化竞争。
4. **品牌重定**：聚贤堂做的是“承接人生场合的一处局”。
5. **定位拆解**：馆解决进入门槛，味解决产品记忆，堂解决体验溢价。
6. **文化支撑**：八大堂、老舍婚宴、炸响铃和堂会礼数让定位有出处。
7. **体验落地**：产品、空间、视觉、服务话术都围绕“入席感”组织。
8. **商业路径**：先跑通小馆模型，再扩展饭庄和堂会，避免第一阶段过重。

## 正式 PPT 制作注意事项

- 每页只保留一个主判断，不要把解释、证据和结论堆在同一页。
- 市场页要先讲“为什么有机会”，不要一上来讲历史。
- 文化页要讲“如何转译到今天”，不要只讲资料。
- 产品页要避免菜单罗列，必须服务“一招、一馆、一街”。
- 空间页要服务“入堂六步”，不要只是 moodboard。
- 视觉页要落到门头、菜单、餐具、礼盒、社媒头像等真实触点。
- 业态页要明确 V1 是当下验证模型，V2/V3 是品牌资产沉淀后的延伸。
