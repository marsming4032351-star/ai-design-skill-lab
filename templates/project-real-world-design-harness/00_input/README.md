# 00_input：现实输入

这里存放 Human Sensor 从真实世界带回来的原始输入。

## 应该放什么

- 现场照片、视频、录音、手绘草图。
- 尺寸、光线、动线、声音、气味、材质、设备和维护条件记录。
- 甲方访谈、经营目标、预算、时间表、限制条件。
- 用户观察、街区记录、竞品走访、社媒评论摘录。

## 不能做什么

- 不要直接放最终设计方案。
- 不要把 AI 生成的效果图当作现场依据。
- 不要只收集“好看的参考图”，忽略场地和人群。
