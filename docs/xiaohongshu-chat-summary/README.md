## 导出方式

使用真实浏览器打开 `index.html`，按 `#card-01` 到 `#card-05` 逐张截图导出到 `cards/`。

导出后检查：

- PNG 文件存在。
- 每张图片尺寸为 1080 x 1350。
- 中文清晰可读。
- 卡片内容与 `post-copy.md` 一致。
