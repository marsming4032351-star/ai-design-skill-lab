# HTML PPT：AI Design Skill Lab 项目分享

这是一份用于项目分享的横向 HTML deck，主题是：

> AI Design Skill Lab：把 AI 设计从一次性输出变成可复用系统

## 文件说明

- `index.html`：可直接在浏览器打开播放的 12 页 HTML PPT。
- `run-notes.md`：内容来源、页表和自检记录。
- `exports/`：预留导出目录，可放封面图、关键页截图或 PDF。

## 页面结构

1. 封面：项目一句话定位
2. 为什么需要这个仓库
3. 项目目标：Design Data Factory / AI Design Skill Lab
4. 四层结构
5. Pipeline：资产生产链路
6. Harness：任务生命周期
7. 当前边界：现在独立，未来打通
8. Output Skill：输出规范
9. 一次任务的完整流动
10. 目录地图
11. 对团队的价值
12. 总结

## 使用方式

打开：

```bash
open docs/ai-design-skill-lab-project-share/index.html
```

播放：

- 右箭头 / 空格：下一页
- 左箭头：上一页
- Home：第一页
- End：最后一页

## 规格

- 比例：16:9
- 风格：瑞士国际主义风 + 轻量信息图
- 字体：系统中文无衬线
- 适用场景：投屏分享、项目介绍、给新人解释仓库结构
